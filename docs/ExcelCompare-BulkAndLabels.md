# Excel比較機能 説明書（一括変換機能とラベル機能）

本書は、Excel比較ページの「一括変換機能」と「ラベル（カテゴリ）機能」について解説します。Blazor Server版 MesWEB の `ExcelCompare.razor`（RCL）で提供される機能です。

## 対象ページ

- URL: `/excel-compare`
- レンダリング: InteractiveServer（Blazor Server）

---

## 1. 一括変換機能

### 概要

現在のセル対応表に含まれる「ブック名（ファイル名）」または「シート名」を、指定ルールに基づき一括で置き換える機能です。大量の項目を手作業で更新する負担を削減します。

### 起動方法

- 画面の「セル対応表の設定」内のツールバーから「一括変換」ボタン（アイコン: ↔）をクリック
- または右下のフローティングツールバーの「一括変換」ボタンをクリック

### 操作手順

1. モーダルで「変換対象」を選択（ラジオボタン）
   - ブック1のファイル名のみ
   - ブック2のファイル名のみ
   - ブック1とブック2のファイル名の両方
   - ブック1のシート名のみ
   - ブック2のシート名のみ
2. 変換元の値を選択/入力
   - ファイル名は候補リストから選択（現在のセル対応表やアップロード済みファイルから候補が表示）
   - シート名は候補（`datalist`）または手入力
3. 変換先の値を選択/入力
   - ファイル名の場合はアップロード済みファイルから選択
   - シート名の場合は手入力または候補から選択
4. 影響件数を確認
   - モーダル内に「影響を受ける項目数」が表示されます
5. 「変換実行」をクリック

### 注意点

- 指定した「変換元」に完全一致するものだけが対象になります（大文字小文字は区別しない）
- 変換対象が見つからない場合、影響件数は 0 のままです
- セルアドレス（例: `A1:A10`）や数式には影響しません（ブック名・シート名のみ）
- 置換後は確認の上、テンプレートへ保存すると次回の再利用が容易です

### 例

- 変換対象: ブック1のファイル名
- 変換元: `old.xlsx`
- 変換先: `new.xlsx`
- 結果: セル対応表内の「ブック1ファイル名が old.xlsx」の項目が一括で `new.xlsx` に置換されます

## 2. ラベル（カテゴリ）機能

### 概要

テンプレートを「ラベル」で分類し、装置別・プロジェクト別・用途別などで整理できる機能です。テンプレートの検索性/再利用性が向上します。

### 関連UI

- 「テンプレート管理」セクション
  - ラベル（カテゴリ）選択ドロップダウン
  - 保存されたテンプレートドロップダウン
  - 「読み込み」「削除」「保存/上書き」ボタン
- ラベルの作成/編集/削除は専用ページで管理
  - URL: `/cell-mapping/templates`

### 使い方

1. ラベルでテンプレートを絞り込み
   - 「ラベル（カテゴリ）」ドロップダウンから選択
   - 「-- すべて表示 --」で全テンプレートを表示
   - 「未分類」でラベル未設定のテンプレートを表示
2. テンプレートを読み込み
   - 「保存されたテンプレート」から選択→「読み込み」クリック
   - セル対応表が復元されます（ファイル名は自動マッチング）
3. テンプレートを保存/上書き
   - 新規保存: テンプレート名を入力→「保存」ボタン
   - 上書き保存: 既存テンプレート選択状態で「上書き」ボタン
   - 保存時にラベルを選択可能（未選択は「未分類」）

### ポイント

- テンプレートには「ファイル名・シート名・セルアドレス・数式・小数桁数・許容誤差」が保存されます
- 「保存」ボタンは状況に応じて表示が「保存」/「上書き」に自動切替
- ファイル未アップロードでもテンプレートは読み込めます（ファイル名ベースで自動割り当て）

### ラベル例

- 装置A、装置B、装置C
- プロジェクトX、プロジェクトY
- 日次チェック、週次チェック
- 製品A、製品B

## 3. セル対応表の並び替え（マッピング入れ替え）

### 概要

セル対応表の各項目（マッピング）は、ドラッグ＆ドロップで自由に並び替えできます。比較結果の出力順やCSVの行順に影響します。

### 操作方法

- 左端のハンドル（≡）をドラッグして上下に移動
- 右下フローティングツールバーの⬆️/⬇️でリストの先頭/末尾へジャンプ
- 「すべて展開/すべて折りたたむ」で編集しやすくしてから並べ替え

### 補助機能（一括操作）

- 一括変換（ブック・シート名の置換）と併用可能
- 展開状態でも折りたたみ状態でも並び替え可

### 注意点

- 並び順はテンプレート保存時にそのまま保存されます
- 誤操作防止のため、ドラッグ中は項目の枠線や影で位置が分かりやすく表示されます
- 大量の項目がある場合は、フローティングボタンで移動を補助してください

## トラブルシューティング

### 一括変換が効かない
- 変換元の値が正しく入力されていない（余計な空白や全角/半角の違い）
- 対象の項目が存在しない→影響件数を確認

### ラベルでテンプレートが表示されない
- 該当ラベルにテンプレートがない
- 「未分類」に入っている可能性あり
- データベース接続エラー→サーバーログ確認

### 上書き保存で意図しない置換が起きた
- 上書きは既存テンプレートを完全置換します
- ボタン表示が「上書き」になっているか必ず確認
