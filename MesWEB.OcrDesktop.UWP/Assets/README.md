# Assets フォルダについて

このフォルダには、以下のUWPアプリケーションのアセット画像を配置する必要があります：

- LockScreenLogo.scale-200.png (400 x 400 px)
- SplashScreen.scale-200.png (1240 x 600 px)
- Square150x150Logo.scale-200.png (300 x 300 px)
- Square44x44Logo.scale-200.png (88 x 88 px)
- Square44x44Logo.targetsize-24_altform-unplated.png (24 x 24 px)
- StoreLogo.png (50 x 50 px)
- Wide310x150Logo.scale-200.png (620 x 300 px)

これらの画像は、Visual Studioで新しいUWPプロジェクトを作成すると自動的に生成されます。
または、任意の画像編集ソフトウェアで作成できます。

一時的には、単色の画像やプレースホルダー画像を使用してもビルドできます。
